Purpose

The goal of this site is make easyer preparetion process for rafting, where user can orend or buy everything what she/he need for that and to not worry about it anymore. This site is needed because preparetion processs for rafting usualy takes relly long time and blew client's minds, especialy if they do that for the first time. We want to make this process easyer right on computer from their's houses! Benefit for the company: the more easyest this process for them, the more clients our company will have!

Audience

The audience of this site is gonna be active peolpe, usualy young, who love extreme and sports. As we are building this site, we need to keep in mind that it's gonna be adventures type of persons who love to spend fun theirs free time. For to inving this kind of person, out content if gonna be clear, understandeble and fun! We are planning to add some yeloow, purple and turquoise color with Open Sans fonts for to make it more attractive for fun makers!
